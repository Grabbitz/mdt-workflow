ดูขั้นตอนการติดตั้งด้านล่าง แล้วนำ URL ที่ได้มาใส่ที่นี่
เพิ่มความปลอดภัย ตั้งให้ตรงกับในสคริปต์

คู่มือการติดตั้ง Google Sheets

  1. สร้าง Google Sheet ใหม่

    ไปที่ sheets.new สร้าง Sheet เปล่า ตั้งชื่อว่า MT Knowledge Base DB

  2. เปิด Apps Script

    ในหน้า Sheet เลือกเมนู Extensions → Apps Script

  3. วางสคริปต์

    ลบโค้ดเดิมออก แล้วคัดลอกสคริปต์ด้านล่างวางลงไป

  4. Deploy เป็น Web App

    กด Deploy → New deployment → เลือก type Web app → ตั้ง Execute as: Me, Who has access: Anyone → กด Deploy

  5. คัดลอก URL

    คัดลอก Web app URL ที่ได้ มาใส่ในช่อง Apps Script Web App URL ด้านบน แล้วกดบันทึก
