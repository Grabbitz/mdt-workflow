# MDT Workflow — Modern Trade Tracker

คลังความรู้และระบบจัดการข้อมูล Modern Trade สำหรับทีม Operations

## Features

- รวบรวม Workflow และวิธีการทำงานของแต่ละ Channel
- เพิ่ม / แก้ไข / ลบข้อมูลได้ตลอดเวลา
- ค้นหาข้อมูลได้ทันที
- รองรับการใช้งานบน Desktop และ Mobile

## Tech Stack

- HTML / CSS / JavaScript (Static Site)
- Hosted on GitHub Pages

## Live Site

[https://grabbitz.github.io/mdt-workflow/](https://grabbitz.github.io/mdt-workflow/)

## การอัปเดตข้อมูล

1. แก้ไขไฟล์ที่ต้องการ (`index.html`, `app.js`, `styles.css`)
2. รันคำสั่ง:

```bash
git add .
git commit -m "update: รายละเอียดที่แก้ไข"
git push origin main
```

3. รอ 1-2 นาที เว็บจะอัปเดตอัตโนมัติ
